# Acrebase Wallet App

A modern wallet application built with Next.js, designed to help users manage balances, track transactions, and access their wallet dashboard in a clean and responsive interface.

## Overview

Acrebase Wallet App is a frontend web application for a digital wallet experience. It provides a polished user interface for:

- Viewing wallet balance and account activity
- Tracking recent transactions
- Accessing wallet-related actions from a centralized dashboard
- Building an easy-to-scale Next.js application foundation for future wallet features

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser at:

```text
http://localhost:3000
```

## Available Scripts
